📖 Описание

Кузнечик — российский стандарт симметричного блочного шифрования (ГОСТ Р 34.12-2015) с размером блока 128 бит и длиной ключа 256 бит. Данная реализация полностью соответствует спецификации и включает все основные режимы работы согласно ГОСТ Р 34.13-2015.
✨ Возможности

    ✅ Полная реализация алгоритма «Кузнечик» согласно ГОСТ Р 34.12-2015

    ✅ 5 режимов шифрования: ECB, CBC, CFB, OFB, CTR

    ✅ Опциональная имитовставка (MAC) для проверки целостности (4 или 8 байт)

    ✅ Генерация ключа из пароля (50 итераций SHA-512)

    ✅ Управление ключами: сохранение, загрузка, просмотр

    ✅ Интерактивное меню с навигацией по номерам

    ✅ Подробный отладочный вывод каждого шага

    ✅ Тестовый цикл для проверки всех режимов

    ✅ Статистика работы программы

    ✅ Кроссплатформенность (Windows, Linux, macOS)

🔐 Режимы шифрования
Режим	Название	Описание	Требует IV
ECB	Electronic Codebook	Простая замена, каждый блок независимо	Нет
CBC	Cipher Block Chaining	Сцепление блоков, XOR с предыдущим шифртекстом	Да (16 байт)
CFB	Cipher Feedback	Обратная связь по шифртексту (поточный)	Да (16 байт)
OFB	Output Feedback	Обратная связь по выходу (поточный)	Да (16 байт)
CTR	Counter	Режим счетчика (поточный)	Да (16 байт)

🛡️ Имитовставка (MAC)

Имитовставка может быть включена или выключена для любого режима шифрования:

    Длина: 4 или 8 байт

    При шифровании: MAC вычисляется от исходных данных и добавляется в конец файла

    При расшифровании: MAC автоматически отделяется и проверяется

    При несовпадении: пользователь предупреждается и может выбрать действие

Ключевые особенности реализации
Компонент	Описание
Умножение в GF(2^8)	Реализовано методом "столбика" с полиномом 0xC3
Преобразование R	Линейное перемешивание со сдвигом
Преобразование L	16-кратное применение R
S-блок	Табличная подстановка по ГОСТ
Генерация ключей	Сеть Фейстеля с 32 итерационными константами
Padding	Специальная схема с 0x80/0x81 (как в Kotlin)
Генерация из пароля	50 итераций SHA-512
